# Désinstaller un programme

## Interface utilisateur

1. Accéder aux paramètres systèmes

2. Cliquer sur la catégorie **Applis**

3. Inscrivez dans la barre de recherche le nom du programme à désinstaller

4. Sélectionner le programme et cliquer sur le bouton **Désinstaller**

5. Une fenêtre de confirmation apparaitra, Cliquer sur le bouton **Désinstaller** dans la fenêtre de confirmation
